Example for Inline CSS

Cascading Style Sheet is a style language that defines layout of HTML documents.CSS properties such as background,border,font,float,display,margin,opacity,padding,text-align,vertical-align,position,color etc.
Inline styles are CSS styles that are applied to one element using the style attribute before closing the opening tag.
Inline styling is useful for applying a unique style to a single HTML element.Inline styles are easy and quick to add.
